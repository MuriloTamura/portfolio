https://murilotamura.github.io/portfolio/

## 📌 Sobre o projeto

Este portfólio foi desenvolvido com o objetivo de:

* 💼 Apresentar meus projetos de forma organizada
* 🚀 Demonstrar minhas habilidades técnicas
* 📖 Mostrar minha evolução como desenvolvedor
* 📬 Facilitar o contato profissional

Ele funciona como uma vitrine dos meus trabalhos e conhecimentos na área de desenvolvimento.

---

## ⚙️ Como funciona

O portfólio é um site estático hospedado via **GitHub Pages**, permitindo acesso rápido e fácil diretamente pelo navegador, sem necessidade de instalação.

### 🧩 Estrutura principal:

* **Home / Apresentação**
  Introdução sobre quem eu sou e meus objetivos

* **Habilidades (Skills)**
  Tecnologias e ferramentas que utilizo

* **Projetos**
  Exibição dos projetos que desenvolvi

* **Contato**
  Links para redes e formas de contato
